* Documentation: https://pysecretary.readthedocs.io.

Examples
--------
.. code-block:: python

    import os
    from pysecretary import pysecretary

    os.environ["PYSECRETARY"] = "test"
    value = pysecretary.get("env://PYSECRETARY")
    print(value)

    value = pysecretary.get("env://PATH")
    print(value)
